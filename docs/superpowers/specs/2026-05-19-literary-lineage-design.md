# 雅致吊线图 (Literary Lineage Mode) 设计文档

**Status: IMPLEMENTED (2026-05-19)**

## 1. 背景与目标 (Background & Goals)
在 1.0 版本发布后，系统引入了“紧凑模式”以在有限空间内展示高密度血脉。最初的实现仅是简单的卡片隐藏和占位压缩，视觉上较为生硬。

**目标**：将基础的紧凑模式重塑为具备书法神韵、文化质感的“雅致吊线图”，还原传统木刻本族谱的视觉体验，同时增强血脉传承的象征意义。

## 2. 核心设计 (Core Design)

### 2.1 朱砂血脉结 (The Vermilion Knot)
*   **设计逻辑**：在姓名垂直排列的顶端与血脉连线交汇处，放置一个半径为 `2.2px` 的朱砂红圆点 (`#b33939`)。
*   **象征意义**：象征血脉的传承节点与生命力的汇聚。
*   **交互**：悬停时红点放大（scale 1.4），增强触觉反馈。

### 2.2 纵向行气 (Calligraphic Rhythm)
*   **字体选择**：强制使用 `Noto Serif SC` (思源宋体)，字重 `700`，颜色采用深墨色 (`#2f2318`)。
*   **动态字间距**：
    *   2字名：`dy: 28px` (舒展)
    *   3字名：`dy: 22px` (匀称)
    *   4字+名：`dy: 18px` (紧凑)
*   **重心微调**：文字整体相对于占位中心略微上移，为下方留出视觉余白。

### 2.3 金石交互 (The Seal Impression)
*   **选中态**：在姓名背后渲染一个极淡的朱砂色矩形 (`#b33939`, opacity 0.08)，模拟宣纸上的印章盖印效果。
*   **悬停态**：姓名颜色切换为古铜金 (`var(--accent-amber)` / `#a96e35`)，模拟金石流光的质感。

## 3. 技术实现 (Technical Implementation)

### 3.1 状态与逻辑
*   **开关**：`PublicationSettings.showCard` (boolean)。
*   **布局参数**：`layout.ts` 中紧凑模式占位设为 `32x110px`，确保垂直排版有足够空间。

### 3.2 渲染方案
*   组件：`PersonCardSvg.vue`。
*   利用 SVG `<tspan>` 的 `dy` 属性实现动态纵向间距。
*   使用 CSS Transition 确保颜色、缩放等状态切换平滑。

## 4. 评估与自检 (Self-Review)
- **空间效率**：相比标准模式，垂直空间利用率提升约 1.8 倍。
- **一致性**：在所有主题下均保持“墨韵朱砂”的基础调性，仅通过变量适配背景。
- **扩展性**：支持 `fontScale` 比例缩放，行气算法自动适配。
