# 示例数据

本目录包含两个预生成的族谱测试数据，可直接导入系统体验完整功能。

| 文件 | 人数 | 说明 |
|------|------|------|
| `performance-test-100-persons.json` | 100 人 | 小型家族，适合快速体验 |
| `performance-test-500-persons.json` | 500 人 | 中型家族，测试性能与缩放 |

## 导入方式

1. 启动系统，登录后进入 Dashboard
2. 点击「导入族谱数据」
3. 选择本目录下的 JSON 文件
4. 系统会自动解析人物、家庭关系并生成可视化树形图

## 数据说明

- 人物姓名、生卒年月为随机生成，仅供测试
- 包含完整的家庭关系（配偶、子女、父母）
- 不含真实个人信息
